# 视频 键盘 鼠标

## **快捷键**
在v1.4.5版本后，支持自定义快捷键,找到shortcuts配置项目, 其中key为web ui上的按钮名称，value为具体的按键。
```
sudo -s
vim /mnt/exec/release/config/app.json
## 默认配置如下：
"shortcuts": {
    "Ctrl+Alt+Del": ["ControlLeft","AltLeft","Delete"],
    "Alt+Tab": ["AltLeft","Tab"],
    "Alt+F4": ["AltLeft","F4"],
    "Alt+Enter": ["AltLeft","Enter"],
    "Ctrl+W": ["ControlLeft","KeyW"]
}
```
